# Sistema de Inventario em Python

Este projeto agora usa apenas Python.

## Stack

- Interface: `Streamlit`
- Regras de negocio e acesso ao banco: `python_app/services.py`
- Banco de dados SQLite: `data/inventory.db`

## Instalacao

```bash
pip install -r requirements.txt
```

## Execucao

```bash
streamlit run app.py
```
